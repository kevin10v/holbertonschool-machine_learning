## Description

This repository covers **Machine Learning** from the mathematical foundations through deep learning and natural language processing. It is the most advanced track at Holberton School, requiring strong Python and mathematics skills.

## Curriculum

### Math Foundations
| Directory | Topic |
|-----------|-------|
| `math/linear_algebra` | Vectors, matrices, dot products, matrix operations |
| `math/plotting` | Matplotlib: lines, scatter, histograms |
| `math/calculus` | Derivatives, integrals, gradient descent |
| `math/probability` | Distributions, Bayes theorem, independence |
| `math/multivariate_prob` | Joint distributions, covariance, correlation |
| `math/bayesian_prob` | Bayesian inference, likelihood, priors |
| `math/convolutions_pooling` | CNNs math: convolutions and pooling |

### Supervised Learning
| Directory | Topic |
|-----------|-------|
| `supervised_learning/classification` | Binary & multiclass classification, logistic regression |
| `supervised_learning/regression` | Linear & polynomial regression |
| `supervised_learning/optimization` | Adam, momentum, batch normalization |
| `supervised_learning/error_analysis` | Bias/variance, confusion matrix |
| `supervised_learning/regularization` | L1, L2, dropout |
| `supervised_learning/keras` | High-level neural networks with Keras |
| `supervised_learning/cnn` | Convolutional Neural Networks |
| `supervised_learning/deep_cnns` | ResNet, DenseNet, Inception |
| `supervised_learning/RNNs` | LSTM, GRU, time series |
| `supervised_learning/word_embeddings` | Word2Vec, GloVe, FastText |
| `supervised_learning/transformer_apps` | BERT, GPT fine-tuning |
| `supervised_learning/object_detection` | YOLO, object detection |
| `supervised_learning/nlp_metrics` | BLEU, ROUGE, perplexity |

### Unsupervised Learning
| Directory | Topic |
|-----------|-------|
| `unsupervised_learning/clustering` | K-Means, DBSCAN, hierarchical |
| `unsupervised_learning/dimensionality_reduction` | PCA, t-SNE |
| `unsupervised_learning/hmm` | Hidden Markov Models |
| `unsupervised_learning/hyperparameter_tuning` | Grid search, random search, Bayesian optimization |
| `unsupervised_learning/autoencoders` | Vanilla, variational, sparse autoencoders |
| `unsupervised_learning/gan` | Generative Adversarial Networks |

### Reinforcement Learning
| Directory | Topic |
|-----------|-------|
| `reinforcement_learning/q_learning` | Q-table, epsilon-greedy |
| `reinforcement_learning/deep_q_learning` | DQN with Atari games |
| `reinforcement_learning/policy_gradients` | REINFORCE algorithm |

## Learning Objectives

- Implement neural networks from scratch using NumPy
- Build and train deep learning models with TensorFlow/Keras
- Apply CNNs, RNNs, Transformers to real-world problems
- Use unsupervised learning for clustering and dimensionality reduction
- Implement reinforcement learning algorithms
